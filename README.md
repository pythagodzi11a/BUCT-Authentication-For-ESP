# BUCT 校园网认证 for ESP

rt,用esp32或者8266进行校园网认证。

---

首先不考虑偷电等保持路由器在线的方案。而在自动认证的几个方案中，要么使用单独的硬件,例如linux开发板，要么就要开机运行等。

能跑linux的开发板，且不需要交叉编译等超级臭超级长的方案不算特别便宜。

最便宜的当然是esp32/8266,因此尝试做了移植。
